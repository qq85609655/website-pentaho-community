Overview

Here you'll discover how easy and simple it is to create or update translations already in use. You can translate all resources from English to other languages, and propose new ones that aren't yet available.
Check out what's already translated.

Contributing to the translation process is as simple as forking the Webdetails repository created for that purpose, and propose a new language or change one of the available ones.

The language packs apply to both Enterprise and Community Edition, and are maintained by community members. The packs are not officially supported by Pentaho.

Workflow (using GIT)

01

Fork the Webdetails' repository PentahoLanguagePacks. To merge your changes, please issue a pull request.

02

Save all files using UTF8 encoding.

03

Do not bother to edit the supported_languages.properties files, their contents will be ignored by the language pack installer.

04

The information in the file data/${languageCode}/metadata.json will be used by the installer to patch Pentaho with your translation, so please make sure the fields languageCode and language are consistent with your modifications.

05

You are now doing translation work for CE using Git. This is great, thanks!

Marketplace in Pentaho 6.1

If you're using the Community Edition of Pentaho 6.1 you don't have to worry about this - Marketplace comes with it and it's ready to be used.
If you have Pentaho 6.1 Enterprise Edition and go to Marketplace Public Site to try to download the Marketplace plugin, you'll see it's not there.
The reason for that? It's an OSGi plugin and you already have it, but it comes disabled by default. Here are the instructions on how to enable it:

  1. Edit the file 
    pentaho-solutions/system/karaf/etc/org.apache.karaf.features.cfg
  2. Find the key "featuresBoot" and add the feature "pentaho-marketplace". That line will then be something like:
    featuresBoot=config,management,kar,cxf,camel,camel-blueprint,camel-stream,pentaho-server,mongolap,pentaho-metaverse,pdi-dataservice-client,pentaho-yarn-obf,pentaho-marketplace

Restart your server, and you're done!
