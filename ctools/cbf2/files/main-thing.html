There are 4 main concepts here:

  • Core images;
  • Core containers;
  • Project images;
  • Project containers.

These should be straightforward to understand if you're familiar with Docker , but in a nutshell there are two fundamental concepts: images and containers. An image is an inert, immutable file.

The container is an instance of an image, and it's a container that will run and allow us to access the Pentaho platform
