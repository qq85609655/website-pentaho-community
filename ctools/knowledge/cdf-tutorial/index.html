CDF TUTORIAL

Learn Dashboards Framework

The Community Dashboard Framework (CDF) is a project that provides an easy to use and powerful dashboard framework on top of the Pentaho BI server. As the main developers of this community project, we understand that sometimes is not easy to achieve exactly what we want. To allow users to get the full benefit of CDF we've prepared a set of documentation - a step by step tutorial from "how to start" to the hidden features of this framework.

CDF Tutorial is supported by Pentaho versions 5.0 and above. If you are running an older version please consider upgrading before purchasing CDF Tutorial.
