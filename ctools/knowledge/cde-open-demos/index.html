Open Demos

Open Demos are a set of dashboards created by the Webdetails team using CDE Editor. As the main developers of the CDE Editor, we understand that it's not easy to plunge into a completely new work logic. So we allow you to edit the dashboards and thus discover and learn the tricks and best practices that are behind each of these dashboards.

All demos bundled in CDE Open Demos are supported by Pentaho versions 5.0 and above. If you are running an older version please consider upgrading before purchasing CDE Open Demos.
