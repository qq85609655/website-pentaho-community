Quickstart

One of the most difficult things in CBF is to get it up and running. In order to make things simpler, we prepared a quickstart bundle.

Download the quickstart bundle here and unzip it.

Create your environment's configuration file

Copy build-pedro.properties to build-yourname.properties and edit it according to your computer.

Download apache tomcat

Download the .zip or .tgz file from Apache page and uncompress it into the CBF Quickstart directory.

Checkout Pentaho from SVN

After checking this out, checkout Pentaho from SVN:

svn co svn://source.pentaho.org/svnroot/bi-platform-v2/tags/3.8.0-stable pentaho-3.8

Run CBF

Use the command: 

ant -Dproject=client -Denv=yourname dist-clean all run
