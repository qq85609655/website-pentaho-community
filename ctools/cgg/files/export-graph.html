Export Graph

Call the URL: http://<pentaho server url>/pentaho/content/cgg/draw?script=<path to the script>, where <path to the script> is, e.g., /steel-wheels/dashboards/Test.js. It means that you want to export the chart called Test, from a Dashboard saved inside the folder dashboards, in your steel-wheels solution.

Don't forget to export the chart before saving the Dashboard. Otherwise, it will not work.


Optionally, you can add some extra parameters to the request. If your chart definition expects a customerNumber as a parameter, simply append paramcustomerNumber= to the query string.

You can also change the output to return either PNG or SVG files (using outputType=png or outputType=svg in the query string.
